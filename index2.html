How to git

  1. cd the directory you need
  2. ls to see what is in the directory
  3. clone the depository
  4. git clone "paste the link to the clone"
  5. cd the depository folder and move you work to that folder
  6. git add . to track the files
  7. git commit -m "add what changes you did"
  8. git push origin master to push the changed files into your repository
